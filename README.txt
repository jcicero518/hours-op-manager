== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the `hours-op-manager` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Upload a CSV
4. Use shortcode `[hours_op_manager]` or place `<?= do_shortcode('[hours_op_manager]'); ?>` in your templates
5. Listen to some good melodic death metal

== Customize ==

You can customize the output of the shortcode with these parameters:

*   tdWidth - width of <td>, defaults to 234
*   headerTag - Tag to use for header, defaults to <h4>
*   headerText - Text for header, defaults to Hours of Operation
*   subheadTag - Tag to use for subhead, defaults to <p>
